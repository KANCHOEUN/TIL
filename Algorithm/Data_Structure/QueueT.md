# Queue\<T>

: deque\<T> 컨테이너를 **선입선출 ( First In First Out )** 로 저장하는 방식으로 변환하는 어댑터 클래스 템플릿

: queue 헤더에 정의되어 있음



1. 큐의 **뒤** 에서만 원소 **추가**
2. 큐의 **앞** 에서만 원소 **제거**



- T& front()

  : 큐가 비어있을 경우, 반환되는 값 정의되어 있지 않음

  : 큐가 비어있지 않을 경우, 첫 번째 원소 ( 큐의 앞 원소 ) 가 반환

- T& back()

  : 큐가 비어있을 경우, 반환되는 값 정의되어 있지 않음

  : 큐가 비어있지 않을 경우, 마지막 원소 ( 큐의 뒤 원소 ) 가 반환

- void push(const T& obj)

  : 큐의 끝에 obj의 복제본 추가

  : 기반 컨테이너의 push_back 멤버 호출하여 수행

- void push(T&& obj)

  : 큐의 끝에 obj를 이동하여 추가

  : 기반 컨테이너의 우측값 참조 매개변수 버전의 push_back() 멤버 호출하여 수행

- void pop()

  : 큐의 첫 번째 원소 삭제

- int size()

  : 큐에 있는 원소의 개수 반환

- bool empty()

  : 큐가 비어있을 경우, true 반환
