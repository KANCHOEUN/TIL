# 벡터 공간 ( Vector Space )

다음 공리를 만족하는 집합 V를 벡터 공간 ( vector space ) 이라 한다.

#### 벡터 공간의 공리 ( axiom )

##### (1) 덧셈에 대한 닫힘 성질 ( closure under addition )

0. 덧셈에 대해 닫힘 : u , v ∈ V → u + v ∈ V 

1. 교환 법칙 ( commutative ) : u + v = v + u
2. 결합 법칙 ( associative ) : (u + v) + w = u + ( v + w )
3. 항등원 ( identity ) : u + 0 = 0 + u = u
4. 역원 ( inverse ) : u + (-u) = (-u) + u



##### (2) 스칼라배에 대한 닫힘 성질 ( closure under scalar multiplication )

0. 스칼라배에 대해 닫힘 : u ∈ V → a u ∈ V

1. **두 vector 와 한 scalar 간**의 덧셈 / 곱셈 연산 : a (u + v) = a u + a v
2. **두 scalar 와 한 vector 간**의 덧셈 / 곱셈 연산 : (a + b) u = a u + b u

3. a ( b u ) = ( a b ) u
4. 항등원 ( identity ) : 1 u = u
