#### 2.3.1 리스트 만들기

```python
리스트명 = [요소1, 요소2, 요소3, ... ]
```

> 비어 있는 리스트는 a = list() 로 생성할 수도 있음



#### 2.3.2 리스트의 인덱싱과 슬라이싱

##### (1) 리스트 인덱싱이란?

```python
>>> a = [1, 2, 3]
>>> a
[1, 2, 3]

>>> a[0] + a[2]
4

>>> a[-1]
3
```



> ###### 이중 리스트
>
> ```python
> >>> a = [1, 2, 3, ['a', 'b', 'c']]
> >>> a[-1]
> ['a', 'b', 'c']
> >>> a[-1][0]
> 'a'
> ```

> ###### 삼중 리스트
>
> ```python
> >>> a = [1, 2, ['a', 'b', ['Life', 'is']]]
> >>> a[2][2][0]
> 'Life'
> ```



##### (2) 리스트 슬라이싱이란?

```python
>>> a = [1, 2, 3, 4, 5]
>>> a[0:2]
[1, 2]
```



> ##### 중첩된 리스트에서 슬라이싱하기
>
> ```python
> >>> a = [1, 2, 3, ['a', 'b', 'c'], 4, 5]
> >>> a = [2:5]
> [3, ['a', 'b', 'c'], 4]
> >>> a[3][:2]
> ['a', 'b']
> ```



#### 2.3.3. 리스트 연산하기

##### (1) 리스트 더하기

```python
>>> a = [1, 2, 3]
>>> b = [4, 5, 6]
>>> a + b
[1, 2, 3, 4, 5, 6]
```



##### (2) 리스트 곱하기

```python
>>> a = [1, 2, 3]
>>> a * 3
[1, 2, 3, 1, 2, 3, 1, 2, 3]
```



##### (3) 리스트 길이 구하기

```python
>>> a = [1, 2, 3]
>>> len(a)
3
```

: len 함수는 문자열, 리스트 외에 튜플과 딕셔너리에도 사용할 수 있는 함수



#### 2.3.4. 리스트의 수정과 삭제

##### (1) 값 수정하기

```python
>>> a = [1, 2, 3]
>>> a[2] = 4
>>> a
[1, 2, 4]
```



##### (2) del 함수로 리스트 요소 삭제하기

```python
>>> a = [1, 2, 3]
>>> del a[1]
>>> a
[1, 3]
```

: del 함수는 파이썬이 자체적으로 가지고 있는 삭제 함수

```python
del 객체	# 객체란 파이썬에서 사용되는 모든 자료형을 말함
```



#### 2.3.5. 리스트 관련 함수들

##### (1) 리스트 요소 추가 ( append )

```python
>>> a = [1, 2, 3]
>>> a.append(4)
>>> a
[1, 2, 3, 4]

# 리스트에 리스트를 추가한 경우
>>> a.append([5, 6])
>>> a
[1, 2, 3, 4, [5, 6]]
```



##### (2) 리스트 정렬 ( sort )

```python
>>> a = [4, 1, 3, 2]
>>> a.sort()
>>> a
[1, 2, 3, 4]

# 문자 또한 알파벳 순서로 정렬할 수 있음
>>> a = ['a', 'c', 'b']
>>> a.sort()
>>> a
['a', 'b', 'c']
```



##### (3) 리스트 뒤집기 ( reverse )

```python
>>> a = ['a', 'c', 'b']
>>> a.reverse()
>>> a
['b', 'c', 'a']
```



##### (4) 위치 반환 ( index )

```python
>>> a = [1, 2, 3]
>>> a.index(3)
2
>>> a.index(0)
Traceback (most recent call last):
  File "<pyshell#6>", line 1, in <module>
    a.index(0)
ValueError: 0 is not in list
```

: 리스트에 존재하지 않을 경우 값 오류 ( ValueError ) 발생



##### (5) 리스트 요소 삽입 ( insert )

```python
>>> a = [1, 2, 3]
>>> a.inser(0, 4)
>>> a
[4, 1, 2, 3]
```

: insert(a, b) 는 리스트의 a 번째 위치에 b를 삽입하는 함수



##### (6) 리스트 요소 제거 ( remove ) / 리스트 요소 끄집어내기 ( pop )

```python
# remove
>>> a = [1, 2, 3, 1, 2, 3]
>>> a.remove(3)
>>> a
[1, 2, 1, 2, 3]

# pop
>>> a = [1, 2, 3]
>>> a.pop(1)
2
>>> a
[1, 3]
```

: remove(x) 는 리스트에서 첫 번째로 나오는 x를 삭제하는 함수

: pop()은 리스트의 맨 마지막 요소를 반환하고 그 요소는 삭제하는 함수



##### (7) 리스트 포함된 요소 개수 세기 ( count )

```python
>>> a = [1, 2, 3, 1]
>>> a.count(1)
2
```



##### (8) 리스트 확장 ( extend )

```python
>>> a = [1, 2, 3]
>>> a.extend([4, 5])
>>> a
[1, 2, 3, 4, 5]
>>> b = [6, 7]
>>> a.extend(b)	# a + b 와 동일
>>> a
[1, 2, 3, 4, 5, 6, 7]
```
