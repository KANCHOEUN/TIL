### 2.6 집합 자료형 ( Set )

#### 2.6.1 집합 자료형 만들기

: set 키워드를 이용

```python
# set() 괄호 안에 리스트 입력
>>> s1 = set([1, 2, 3])
>>> s1
{1, 2, 3}

# set() 괄호 안에 문자열을 입력
>>> s2 = set("Hello")
>>> s2
{'e', 'H', 'l', 'o'}
```



#### 2.6.2 집합 자료형의 특징

1. **중복을 허용하지 않는다.**

2. **순서가 없다. ( Unordered )**

   : 리스트나 튜플은 순서가 있음 ( ordered ) →  인덱싱을 통해 자료형의 값을 얻을 수 있음

   : set 자료형과 dictionary는 순서가 없음 ( unordered ) → 인덱싱으로 값을 얻을 수 없음
   
     따라서 set 자료형에 저장된 값을 인덱싱으로 접근하려면 다음과 같이 리스트나 튜플로 변환해야 함
   
   ```python
   >>> s1 = set([1, 2, 3])
   >>> l1 = list(s1)
   >>> l1
   [1, 2, 3]
   
   >>> t1 = tuple(s1)
   >>> t1
   (1, 2, 3)
   >>> t1[0]
   1
   ```



#### 2.6.3 교집합, 합집합, 차집합 구하기

```python
>>> s1 = set([1, 2, 3, 4, 5, 6])
>>> s2 = set([4, 5, 6, 7, 8, 9])
```



##### (1) 교집합 ( & , intersection 함수 )

```python
>>> s1 & s2	# s1.intersection(s2)
{4, 5, 6}
```



##### (2) 합집합 ( | , union 함수 )

```python
>>> s1 | s2	# s1.union(s2)
{1, 2, 3, 4, 5, 6, 7, 8, 9}
```



##### (3) 차집합 ( - , difference 함수 )

```python
>>> s1 - s2	# s1.difference(s2)
{1, 2, 3}
>>> s2 - s1	# s2.difference(s1)
{8, 9, 7}
```



#### 2.6.4 집합 자료형 관련 함수들

##### (1)  값 1개 추가하기 ( add )

```python
>>> s1 = set([1, 2, 3])
>>> s1.add(4)
>>> s1
{1, 2, 3, 4}
```



##### (2) 값 여러 개  추가하기 ( update )

```python
>>> s1 = set([1, 2, 3])
>>> s1.update([4, 5, 6])
>>> s1
{1, 2, 3, 4, 5, 6}
```



##### (3) 특정 값 제거하기 ( remove)

```python
>>> s1 = set([1, 2, 3])
>>> s1.remove(2)
>>> s1
{1, 3}
```
