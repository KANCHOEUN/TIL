# 1. Python 이란 ?

**인터프리터 언어**
: 한 줄씩 소스 코드를 해석해서 **그때그때 실행**해 **결과를 바로 확인**할 수 있는 언어



**파이썬**으로

1. 할 수 있는 일

   - 시스템 유틸리티 ( 컴퓨터 이용에 도움이 되는 여러 소프트웨어 ) 제작

     : 파이썬은 운영체제의 시스템 명령어들을 이용할 수 있는 각종 도구를 갖추고 있음

   - GUI (Graphic User Interface) 프로그래밍

     : 윈도우 창처럼 화면을 보며 마우스나 키보드로 조작할 수 있는 프로그램

     : 파이썬 프로그램을 설치 시에 같이 설치되는 기본 모듈인 Tkinter 를 이용해 만들 수 있음 

   - C/C++와의 결합

     : 다른 언어들과 잘 어울려 결합해서 사용할 수 있어서 파이썬을 접착 (glue) 언어라고 부른다.

   - 웹 프로그래밍

   - 수치 연산 프로그래밍

     : 파이썬에는 NumPy ( C로 작성된 모듈 ) 이라는 수치 연산 모듈이 제공됨

   - 데이터베이스 프로그래밍

     : Sybase, Informix, Oracle, MySQL 등의 데이터베이스에 접근할 수 있게 해주는 도구들을 제공

   - 데이터 분석, 사물 인터넷

     : Pandas 라는 모듈을 이용하여 데이터 분석

   

2. 할 수 없는 일

   - 시스템과 밀접한 프로그래밍 영역
   - 모바일 프로그래밍



- 대화형 인터프리터 : 파이썬 쉘 ( Python Shell )

- 프롬프트 ( prompt ) : 3개의 꺾은 괄호 ( >>> )



- IDLE ( Integrated Development and Learning Environment ) : 프로그램 작성을 도와주는 통합 개발 환경
  - IDLE 쉘 창 ( Shell Window ) : IDLE 에디터에서 실행한 프로그램의 결과가 표시되는 창
  - IDLE 에디터 창 ( Editor Window ) : IDLE 에디터가 실행되는 창



# 2. Python 자료형



### 2.1 숫자형 ( Number )

: 숫자 형태로 이루어진 자료형



#### 2.1.1 숫자형 자료형 종류

##### (1) 정수형 ( Integer )



##### (2) 실수형 ( Floating - point )



##### (3) 8진수 ( Octal ) / 16진수 ( Hexadecimal )

​    : 8진수는 0o 또는 0O ( 숫자 0 + 알파벳 o/O ) 로 시작하고 16진수는 0x 로 시작



#### 2.1.2 숫자형을 활용하는 연산자

##### (1) 사칙연산

```python
>>> a + b
>>> a - b
>>> a * b
>>> a / b
```

( / 는 몫만이 아닌 몫과 나머지를 함께 반환 )



##### (2) x 의 y 제곱

```python
>>> x ** y
```



##### (3) 나머지만 반환

```python
>>> a % b
```



##### (4) 몫만 반환

```python
>>> a // b
```
