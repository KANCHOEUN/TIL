#### 2.8.1 변수란?

```python
# 변수명 = 변수에 저장할 값
```

: C 나 Java 처럼 변수의 자료형을 함게 쓸 필요 없음

: 변수란 객체를 가리키는 것

: 객체란 앞의 자료형들과 같은 것을 의미



```python
>>> a = [1, 2, 3]
>>> id(a)
4303029896
```

: 객체가 자동으로 메모리에 생성되고 변수 a 에 그 객체가 저장된 **메모리의 주소**를 가리키게 됨

: 파이썬 내장함수인 id 함수는 변수가 가리키고 있는 객체의 주소를 리턴함

 

#### 2.8.2 리스트 복사

```python
>>> a = [1, 2, 3]
>>> b = a
```

: a 가 가리키는 대상과 b 가 가리키는 대상이 동일



##### b 변수를 생성할 때 a 와 같은 값을 가지도록 복사하고

##### a 가 가리키는 리스트와는 다른 리스트를 가리키게 하는 방법은 ?



##### (1) [ : ] 이용

: **리스트 전체**를 가리키는 [:] 을 이용

```python
>>> b = a[:]
```



##### (2) copy 모듈 이용

```python
>>> from copy import copy
>>> b = copy(a)
```



#### 2.8.3 변수 만드는 방법

##### (1) 튜플로 만들기

```python
>>> a, b = ('python', 'life')	# 튜플 괄호 생략 가능
>>> (a, b) = 'python', 'life'
```



##### (2) 리스트로 만들기

```python
>>> [a, b] = ['python', 'life']
```



##### (3) 여러 개의 변수에 같은 값 대입

```python
>>> a = b = 'python'

# 이 방법을 이용하여 두 변수의 값을 간단히 바꿀 수 있음
>>> a = 3
>>> b = 5
>>> a, b = b, a
```
