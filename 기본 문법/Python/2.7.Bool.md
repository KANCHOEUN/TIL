### 2.7 불 자료형 ( bool )

[TOC]

#### 2.7.1 불 자료형이란?

: 참 ( True ) 과 거짓 ( False ) 을 나타내는 자료형

: 첫 문자 항상 대문자



#### 2.7.2 자료형의 참과 거짓

: 문자열, 리스트, 튜플, 딕셔너리 등의 값이 비어 있으면 ( " " , [ ] , ( ) , { } ) 거짓

: 0 과 None 은 거짓



#### 2.7.3 불 연산

: bool 이라는 내장 함수를 통해 자료형의 참과 거짓을 식별할 수 있음

```python
>>> bool('python')
True
>>> bool('')
False
```
