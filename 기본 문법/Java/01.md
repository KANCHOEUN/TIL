# 01. 시작



#### 1. JDK 다운로드

###### jdk 내의 bin 디렉토리

: 개발 및 실행에 필요한 여러가지 도구 ( 프로그램 ) 들이 존재

1. Javac.exe : 자바 컴파일러
2. Java.exe : 자바 런처



> ###### 환경 변수 Path
>
> : 윈도우 운영체제가 명령 프롬프트에서 입력된 실행 파일을 찾는 경로의 정보가 됨



> **명령 프롬프트**는 Path 가 알려주는 경로로 가서 실행 파일을 가져다가 실행을 시킴



#### 2. 자바 프로그램의 실행 구조

##### (1) 일반적인 프로그램

![01-(1)](../../Images/01-(1).JPG) 

: 하드웨어 위에서 실행되는 운영체제가 프로그램을 실행시키는 구조



##### (2) 자바 프로그램

![01-(2)](../../Images/01-(2).JPG)  

: 운영체제가 자바 가상머신을 실행시키고, 자바 가상머신이 자바 프로그램을 실행시키는 구조

: 운영체제 입장에서 가상머신의 존재는 알지만 그 위에서 실행되는 프로그램을 모름

: OS에 따라 JVM 이 다르므로 **운영체제에 상관없이 실행 가능**



#### 3. 자바 컴파일러와 자바 바이트코드

##### 자바 컴파일러

: 자바 가상머신이 이해할 수 있는 코드 ( 자바 바이트코드 ) 를 생성

: 바이트 ( 명령어의 크기가 기본적으로 1 Byte ) 코드는 확장자가 class 인 클래스 파일에 담김
