# 02. 변수 ( Variables )

: 데이터의 저장과 참조를 위해 ' 할당된 메모리 공간 ' 에 붙인 이름



1. 어떻게 할당
2. 어떻게 접근



#### 1. 기본 자료형 ( Primitive Data Type )

| 자료형  |  데이터   | 크기 ( Byte ) |  표현 가능 범위   |
| :-----: | :-------: | :-----------: | :---------------: |
| boolean | 참 / 거짓 |       1       |    true, false    |
|  char   |   문자    |       2       |     유니코드      |
|  byte   |   정수    |       1       |    -128 ~ 127     |
|  short  |           |       2       | -32,768 ~ 32,767  |
|   int   |           |       4       | - 약 2억 ~ 약 2억 |
|  long   |           |       8       |                   |
|  float  |   실수    |       4       |                   |
| double  |           |       8       |                   |



> ###### 컴퓨터의 실수 표현에는 오차가 존재

> ###### 정수형 연산은 int 형으로 진행

> float 와 double 사이에서의 자료형 선택 기준은 **정밀도**



#### 2. 참조 타입 ( Reference Type )

: 용도는 세가지 ( 배열 / 클래스 / 인터페이스 )

```java
// class 를 만든다는 것은 Car 라는 데이터 타입을 만든다는 의미
class Car {
    ...
}
```



#### 3. 변수 이름 정하기

1. Java 는 대소문자를 구분
2. 변수의 이름은 숫자로 시작할 수 없음
3. $ 과 _ 이외의 다른 특수 문자 사용 불가
4. 키워드는 변수의 이름으로 사용할 수 없음



> ##### 관습
>
> (1) **클래스** 이름 : **첫 번째 문자부터 대문자**로 시작, 각 단어의 첫 번째 문자만 대문자
>
> (2) **변수, 메소드** 이름 : **첫 단어 이후** 각 단어의 첫 번째 문자는 **대문자**로 시작
>
> (3) **상수** 이름 : 이름 **전체를 대문자**로 표기
